# ☁️ Armazenamento na AWS

## 📖 Sobre

Este módulo apresenta os principais conceitos de armazenamento em nuvem na AWS, com foco no Amazon S3 e nos conceitos relacionados a buckets, objetos, permissões e integração com outros serviços da plataforma.

---

## 🎯 Objetivo

Compreender como a AWS realiza o armazenamento de dados utilizando diferentes modelos, principalmente o armazenamento de objetos através do Amazon S3, além de conhecer conceitos de segurança e gerenciamento de recursos.

---

## 📚 Conteúdos estudados

- Amazon S3 (Simple Storage Service);
- Buckets;
- Objetos e identificadores únicos;
- Hospedagem de sites estáticos;
- Controle de acesso público;
- Bucket Policy;
- ACLs;
- Amazon EBS;
- Volumes;
- Snapshots;
- Criptografia de armazenamento;
- Sistemas de arquivos Linux e Windows.

---

## 🪣 Amazon S3

O Amazon S3 é um serviço de armazenamento de objetos da AWS projetado para oferecer:

- Alta escalabilidade;
- Alta disponibilidade;
- Segurança;
- Armazenamento de grandes volumes de dados.

Os arquivos são armazenados dentro de estruturas chamadas **Buckets**.

---

## 💾 Amazon EBS

O Amazon EBS (Elastic Block Store) fornece volumes de armazenamento em bloco utilizados principalmente com instâncias EC2.

Durante os estudos foram abordados:

- Criação e associação de volumes;
- Verificação de zonas de disponibilidade;
- Formatação de discos;
- Snapshots;
- Criptografia.

---

## 🛠️ Práticas realizadas

- Criação de Bucket no Amazon S3;
- Upload de arquivos;
- Configuração de hospedagem de site estático;
- Configuração de permissões de acesso;
- Criação e gerenciamento de volumes EBS;
- Criação de snapshots;
- Associação de volumes em instâncias EC2.
