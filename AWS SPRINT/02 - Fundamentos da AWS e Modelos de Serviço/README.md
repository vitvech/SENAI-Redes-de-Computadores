# ☁️ Fundamentos da AWS e Modelos de Serviço

## 📖 Sobre

Este módulo reúne os estudos introdutórios sobre a plataforma AWS, abordando os principais conceitos de Cloud Computing, modelos de cobrança, modelos de serviço, infraestrutura global da AWS e boas práticas de arquitetura.

---

## 🎯 Objetivo

Compreender como funciona a infraestrutura da AWS, os modelos de serviço em nuvem, os diferentes modelos de cobrança e os princípios utilizados para projetar arquiteturas eficientes, escaláveis e confiáveis.

---

## 📚 Conteúdos estudados

- Custos em Cloud Computing (CapEx e OpEx)
- Modelos de pagamento (Pay-as-you-go, Savings Plans e Instâncias Reservadas)
- Métricas de cobrança
- Modelos de serviço (IaaS, PaaS e SaaS)
- AWS Pricing Calculator
- Infraestrutura Global da AWS
- Regiões e Availability Zones (AZs)
- Edge Locations e Local Zones
- AWS Well-Architected Framework
- Os seis pilares do Well-Architected Framework

---

## 🛠️ Serviços e recursos abordados

- Amazon EC2
- Amazon S3
- Amazon VPC
- Elastic Load Balancing (ELB)
- Amazon CloudWatch
- AWS Pricing Calculator

---

## 💡 O que aprendi

Durante este módulo compreendi como a AWS estrutura sua infraestrutura global, como funciona a cobrança pelos serviços em nuvem, as diferenças entre os modelos IaaS, PaaS e SaaS e a importância de projetar soluções seguindo as boas práticas do AWS Well-Architected Framework.
