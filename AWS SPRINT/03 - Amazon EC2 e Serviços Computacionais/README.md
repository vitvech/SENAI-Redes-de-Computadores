# 🖥️ Amazon EC2 e Serviços Computacionais

## 📖 Sobre

Este módulo apresenta os principais serviços computacionais da AWS, com foco no Amazon EC2 (Elastic Compute Cloud), abordando conceitos de máquinas virtuais, configuração de instâncias, segurança, armazenamento e escalabilidade.

---

## 🎯 Objetivo

Compreender como a AWS disponibiliza recursos computacionais sob demanda, aprendendo como criar e gerenciar servidores virtuais, configurar recursos de rede, escolher opções de armazenamento e aplicar conceitos de escalabilidade.

---

## 🛠️ Conceitos abordados

- Servidores virtuais na nuvem;
- Configuração de recursos computacionais;

---

## ☁️ Principais serviços

### Amazon EC2

Serviço que permite criar máquinas virtuais na AWS, oferecendo controle sobre sistema operacional, processamento, memória, armazenamento e rede.

### Amazon EBS

Serviço de armazenamento em bloco utilizado como disco virtual para instâncias EC2.

### Amazon EFS

Sistema de arquivos compartilhado que permite acesso aos dados por múltiplas instâncias.

### ECS e EKS

Serviços voltados para execução e gerenciamento de containers.

---

## 💡 O que aprendi

Neste módulo aprendi como funciona a base computacional da AWS, entendendo como servidores tradicionais podem ser migrados para a nuvem utilizando instâncias EC2.

Também compreendi conceitos importantes como AMIs, tipos de instância, grupos de segurança, armazenamento, IPs públicos e escalabilidade.
